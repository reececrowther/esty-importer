# 5 Vertical Modern Frame Pack

Place your PSD files in this folder with these exact names:

- `vertical-1.psd`
- `vertical-2.psd`
- `vertical-3.psd`
- `vertical-4.psd`
- `vertical-5.psd`

Each PSD must have a **Smart Object** layer for the design. Supported layer names: `YOUR DESIGN HERE`, `Design Here`, or `Design` (or set a custom name in the app).
