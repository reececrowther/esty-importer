# Default mockup pack

This pack is used when users choose **Use a mockup pack** in Step 2.

## Required files

Place PSD files in this folder with these exact names:

**Vertical (5):**
- `vertical-1.psd`
- `vertical-2.psd`
- `vertical-3.psd`
- `vertical-4.psd`
- `vertical-5.psd`

**Horizontal (5):**
- `horizontal-1.psd`
- `horizontal-2.psd`
- `horizontal-3.psd`
- `horizontal-4.psd`
- `horizontal-5.psd`

Packs will be expanded and grouped in a future update.
